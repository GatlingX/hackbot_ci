# Hackbot CLI

## Step 1

Install the Hackbot CLI

```bash
pip install hackbot
```

## Step 2


Get a Hackbot API key from the [Hackbot website](https://hackbot.org).

## Step 3

Run the Hackbot CLI on a foundry project.

```bash
python -m hackbot --api-key <your-api-key> --source <path-to-your-project>
```
